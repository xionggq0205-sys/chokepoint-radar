# ADR-001: 瓶颈评分采用1-5分制而非百分比

## 状态
已采纳

## 背景
瓶颈投资法有四个筛选维度（物理必需、寡头垄断、产能刚性、小市值），需要一种评分方式让用户快速判断标的是否符合标准。

## 决策
采用1-5分整数评分，不用百分比或1-10分。

## 理由
1. **颗粒度合适**：瓶颈评分本质是半定量判断，1-5刚好区分"完全不符合(1)"到"完美符合(5)"
2. **可累加**：四维总分5-20分，一眼看出强弱（源杰科技19 vs 中际旭创11）
3. **与雷达图天然匹配**：5个刻度，ECharts默认支持
4. **降低主观偏差**：迫使明确选择，避免"给个72%"这种模糊判断

## 后果
- 同分标的可能结构不同 → 点击维度展开评分依据文本

## 替代方案
- 百分比：虚假精确 | 1-10分：中间值太多 | 三级制：颗粒度不够
